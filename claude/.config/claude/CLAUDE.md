# 必ず守るべきルール
- 不明点がなくなるまでAskUserQuestionで繰り返し質問すること
- コンテキストが70%以上になったら圧縮を提案すること
 
# Plan Mode Rules

## 出力制限
- プラン出力: 最大200ワード
- 各ステップ: ファイル名、コマンド、行番号を含める
- 最後に: 1-3個の具体的な質問で終わる

## 禁止表現
- "needs to be considered"
- "comprehensive approach"
- その他の抽象的フレーズ

## 必須フォーマット
各アクション:
- What: 具体的なファイルまたはコマンド
- Where: ファイルパス:行番号
- Why: 20文字以内
